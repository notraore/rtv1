<h1>RTv1</h1>

usage : `make`<br />
`./rt scenes/[your map]`<br />

<h2>Mouvement :</h2><br />
<ul>
<li>Translate : W, A, S, D</li><br />
<li>Rotate : UP, LEFT, DOWN, RIGHT</li><br />
<li>Lift UP/DOWN : Q, E</li><br />
<li>Pivot : Z, X</li><br />
</ul>
<h2>Introducing to :</h2><br />
<ul>
<li>Anti-Aliasing</li><br />
<li>Multi-light spot</li><br />
<li>Shadow</li><br />
<li>Shiny objects</li><br />
</ul>
